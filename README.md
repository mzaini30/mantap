Tools untuk optimasi CSS, JS, dan gambar-gambar supaya websitenya wuzz wuzz...

## Persiapan

Install Node JS

## Instalasi

Jalankan: `npm i -g mantap`

## Menjalankan

Letakkan file-file CSS, JS, dan gambar di folder `src`

Jalankan di Terminal/Command Prompt: `mantap`
